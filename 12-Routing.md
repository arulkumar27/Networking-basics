# Routing

## What is Routing?

Routing is the process of forwarding packets from one network to another.

---

## Router

A router connects different networks together.

Example:

Home Network → Internet

---

## Types of Routing

### Static Routing

Routes are configured manually.

Advantages:
- Simple
- Predictable

Disadvantages:
- Difficult to manage at scale

---

### Dynamic Routing

Routes are learned automatically.

Protocols:
- RIP
- OSPF
- BGP

Advantages:
- Automatic Updates
- Better Scalability

---

## Routing Table

Stores:

- Destination Network
- Next Hop
- Interface

Command:

route print

or

ip route
