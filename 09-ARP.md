# ARP

ARP stands for Address Resolution Protocol.

Purpose:
Find MAC Address from an IP Address.

---

## Example

IP Address:

192.168.1.10

ARP finds:

MAC Address:

00:1A:2B:3C:4D:5E

---

## ARP Process

1. ARP Request Broadcast
2. Device Responds
3. ARP Table Updated

---

## ARP Table

Stores:

IP Address ↔ MAC Address Mapping

Command:

arp -a
