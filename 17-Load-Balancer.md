# Load Balancer

## What is a Load Balancer?

A load balancer distributes traffic across multiple servers.

---

## Benefits

- High Availability
- Scalability
- Fault Tolerance

---

## Types

### Layer 4 Load Balancer

Works with:

- TCP
- UDP

Example:

AWS Network Load Balancer

---

### Layer 7 Load Balancer

Works with:

- HTTP
- HTTPS

Example:

AWS Application Load Balancer

---

## Algorithms

### Round Robin

Traffic distributed equally.

### Least Connections

Traffic sent to least busy server.

### Weighted Round Robin

Traffic distributed based on server capacity.
