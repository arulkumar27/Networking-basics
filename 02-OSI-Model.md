# OSI Model

OSI stands for Open Systems Interconnection.

The OSI Model consists of 7 layers.

---

## Layer 7 - Application

Protocols:
- HTTP
- HTTPS
- FTP
- SMTP

Purpose:
Provides services to users.

---

## Layer 6 - Presentation

Purpose:
- Encryption
- Compression
- Data Translation

Examples:
- SSL
- TLS

---

## Layer 5 - Session

Purpose:
Manages communication sessions.

---

## Layer 4 - Transport

Protocols:
- TCP
- UDP

Purpose:
Reliable data transmission.

---

## Layer 3 - Network

Protocols:
- IP
- ICMP

Device:
- Router

Purpose:
Routing packets.

---

## Layer 2 - Data Link

Device:
- Switch

Purpose:
MAC Address communication.

---

## Layer 1 - Physical

Examples:
- Fiber Cable
- Ethernet Cable

Purpose:
Physical transmission of bits.
