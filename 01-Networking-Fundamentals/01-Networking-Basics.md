# Networking Basics

## What is Networking?

Networking is the process of connecting multiple devices together so they can communicate and exchange data.

Examples:
- Browsing websites
- Sending emails
- Video calls
- Online gaming
- File sharing

---

## Why Networking is Important

- Resource Sharing
- Internet Access
- Communication
- Remote Access
- Centralized Management

---

## Components of a Network

### Client

A device that requests services.

Examples:
- Laptop
- Mobile Phone
- Desktop

### Server

A device that provides services.

Examples:
- Web Server
- Database Server
- Mail Server

---

## Types of Networks

### LAN (Local Area Network)

Small network within a building.

Examples:
- Home WiFi
- Office Network

### WAN (Wide Area Network)

Large network covering multiple locations.

Example:
- Internet

### MAN (Metropolitan Area Network)

Network covering a city.

### PAN (Personal Area Network)

Personal device network.

Examples:
- Bluetooth
- Mobile Hotspot

---

## Basic Networking Devices

- Router
- Switch
