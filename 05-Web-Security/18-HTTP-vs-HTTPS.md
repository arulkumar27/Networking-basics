# HTTP vs HTTPS

## HTTP

HyperText Transfer Protocol

Port:
80

Characteristics:
- Unencrypted
- Less Secure

---

## HTTPS

HyperText Transfer Protocol Secure

Port:
443

Characteristics:
- Encrypted
- More Secure

---

## Why HTTPS?

Benefits:

- Data Encryption
- Data Integrity
- Authentication

---

## Example

HTTP:

User → Website

Data visible.

---

HTTPS:

User → Encrypted Tunnel → Website

Data protected.
