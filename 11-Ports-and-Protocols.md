# Ports and Protocols

## What is a Port?

A port is a logical communication endpoint used by applications.

Example:

IP Address = House Address

Port Number = Room Number

---

## Types of Ports

### Well-Known Ports

0 - 1023

Examples:
- HTTP
- HTTPS
- SSH

### Registered Ports

1024 - 49151

### Dynamic Ports

49152 - 65535

Used temporarily by applications.
