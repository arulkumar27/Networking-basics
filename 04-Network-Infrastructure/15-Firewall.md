# Firewall

## What is a Firewall?

A firewall controls incoming and outgoing network traffic.

---

## Purpose

- Security
- Access Control
- Threat Prevention

---

## Types of Firewalls

### Stateless Firewall

Checks individual packets.

---

### Stateful Firewall

Tracks connection state.

More secure than stateless firewalls.

---

### Web Application Firewall (WAF)

Protects web applications.

Protects against:
- SQL Injection
- XSS
- Application Attacks

---

## Examples

- Windows Firewall
- pfSense
- Cisco ASA
- Palo Alto
