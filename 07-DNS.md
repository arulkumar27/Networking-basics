# DNS

DNS stands for Domain Name System.

Purpose:
Convert domain names into IP addresses.

---

## Example

google.com

↓

142.250.x.x

---

## DNS Resolution Process

1. Browser Cache
2. OS Cache
3. Recursive Resolver
4. Root Server
5. TLD Server
6. Authoritative Server

---

## Common DNS Records

### A Record

Maps domain to IPv4.

### AAAA Record

Maps domain to IPv6.

### CNAME

Alias Record.

### MX

Mail Server Record.

### TXT

Text Record.

### NS

Name Server Record.
