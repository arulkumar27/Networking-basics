# MAC Address

MAC Address stands for Media Access Control Address.

A unique hardware address assigned to a network card.

---

## Example

00:1A:2B:3C:4D:5E

---

## Characteristics

- Layer 2 Address
- Physical Address
- Usually Permanent

---

## Uses

- Device Identification
- Switching Communication

---

## MAC Address Table

Switches maintain MAC Address Tables.

Purpose:
Forward frames efficiently.
