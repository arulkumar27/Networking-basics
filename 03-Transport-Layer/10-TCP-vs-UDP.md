# TCP vs UDP

## TCP

Transmission Control Protocol

Characteristics:

- Reliable
- Connection Oriented
- Error Checking
- Ordered Delivery

Examples:

- HTTP
- HTTPS
- SSH
- FTP

---

## UDP

User Datagram Protocol

Characteristics:

- Fast
- Connectionless
- No Delivery Guarantee

Examples:

- DNS
- VoIP
- Video Streaming
- Gaming

---

## TCP Three-Way Handshake

Step 1:

Client → SYN

Step 2:

Server → SYN-ACK

Step 3:

Client → ACK

Connection Established

---

## Comparison

| Feature     | TCP | UDP |
|-------------|-----|-----|
| Reliable    | Yes | No |
| Fast        | No | Yes |
| Connection  | Yes | No |
| Ordering    | Yes | No |
