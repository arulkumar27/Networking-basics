# TCP/IP Model

The TCP/IP model is used on the Internet.

---

## Application Layer

Protocols:
- HTTP
- HTTPS
- FTP
- DNS

---

## Transport Layer

Protocols:
- TCP
- UDP

Purpose:
End-to-end communication.

---

## Internet Layer

Protocols:
- IP
- ICMP

Purpose:
Packet routing.

---

## Network Access Layer

Examples:
- Ethernet
- WiFi

Purpose:
Physical network communication.

---

## Example

Opening google.com:

Application → HTTP

Transport → TCP

Internet → IP

Network Access → Ethernet/WiFi
